
# Archlinux dotfiles

This is the backup of my Archlinux config files and packages.
Powered by https://www.chezmoi.io



### Installation

```bash
  sudo pacman -S chezmoi
  chezmoi init https://github.com/freakonaut/dotfiles.git
  chezmoi cd
  ./dot_setup-script.sh
  chezmoi apply
```
      reboot
